# User Guide

This guide explains how to use the Leaderboard App for Instructors (Admins) and Student Assistants.

As an instructor, you have full control over your classes, students, and leaderboard settings.

### 1. Dashboard Management
*   **Create Class**: Click the **"+ New Class"** button in the top right. Enter a descriptive name (e.g., "Web Development A").
*   **Edit Class**: Click the **Edit (Pencil)** icon on a class card to rename it.
*   **Delete Class**: Click the **Trash** icon to permanently delete a class and all its data.

### 2. Managing Students
Inside a class, you have two ways to add students:
*   **Enroll Student**: Click "Enroll Student" to add a single student by Name and Email.
*   **Import CSV**: Click "Import CSV" to upload a bulk list.
    *   *Tip: Use the provided template to ensure your CSV is formatted correctly.*

### 3. Grading & Leaderboard
*   **Award Points**: Hover over a row and click the **Green (+)** button. enter the amount and a reason (e.g., "Good participation").
*   **Penalty**: Click the **Orange (-)** button to deduct points.
*   **Bulk Actions**: Select multiple students using the checkboxes on the left, then click the "Add" or "Penalty" buttons in the top toolbar to grade everyone at once.

### 4. Sharing
*   **Public Link**: By default, leaderboards are public. You can share the link shown in the header (e.g., `/p/your-class-slug`).
*   **Embed**: Click the **Embed** button to get an iframe code snippet. You can paste this into Moodle, Canvas, or Notion to display the live leaderboard there.
*   **Export**: Click the **Export** button to download the current standings as a CSV file for your gradebook.
